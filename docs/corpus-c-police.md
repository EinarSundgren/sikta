# Test Corpus C: Criminal Investigation — The Lindholmen Incident

## About This Corpus

A man found dead in a parking garage in Lindholmen, Gothenburg. Initial report as accident, later reclassified to suspected homicide. Seven documents from the investigation file. Contains 10 planted inconsistencies.

**Language:** Swedish (police report language)
**Extraction challenges:** Contradicting witness statements, temporal impossibilities, forensic evidence vs. witness claims, identity resolution (witnesses refer to same people differently), absent evidence detection.
